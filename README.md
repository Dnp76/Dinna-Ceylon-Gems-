# Dinna Ceylon Gems - Luxury Website

This is a premium, high-performance static website built for a luxury gemstone export brand.

## How to Run
Simply open `index.html` in any modern web browser. No installation or server is required.

## Structure
- **index.html**: Main landing page with cinematic video hero.
- **about.html**: Brand heritage, history, and map.
- **gems.html**: Product catalog with modal details view.
- **export.html**: Services and logistics explanation.
- **gallery.html**: Visual media gallery.
- **contact.html**: Export inquiry form.
- **assets/**: Contains CSS and JS files.

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- FontAwesome (Icons)
- Google Fonts (Playfair Display, Lato)

## Customization
To change images, edit the `src` attributes in the HTML files.
To change colors/fonts, edit the Tailwind config script in the `<head>` of each file or `assets/CSS/style.css`.
